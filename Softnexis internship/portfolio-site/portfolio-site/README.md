# Soft Nexis Internship Task 1 – Portfolio Website

## Project Name: TaskFlow Portfolio

### Features
- 3 Pages: Home, About, Contact
- Responsive Navbar with Hamburger Menu
- Modern CSS using Flexbox and Grid
- Hover Effects and Clean UI

### Deployment Options
#### GitHub Pages
1. Push project to GitHub
2. Go to Settings → Pages
3. Select branch main → root folder
4. Live URL will be generated

#### Netlify
- Drag and drop folder into Netlify Drop

### Tools Used
- HTML5
